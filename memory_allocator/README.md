# Go 内存分配器

## 概述

本项目深入讲解 Go 内存分配器的实现原理，包括 mspan、mcache、mcentral、mheap 等核心组件。

## 核心内容

### 1. 内存管理层次

```
┌─────────────────────────────────────────┐
│                  mheap                  │
│              (全局堆内存)                │
├─────────────────────────────────────────┤
│  ┌───────────┐    ┌───────────┐       │
│  │ mcentral   │    │ mcentral   │ ...  │
│  └───────────┘    └───────────┘       │
└─────────────────────────────────────────┘
            ▲
            │
┌─────────────────────────────────────────┐
│                  mcache                  │
│              (每P本地缓存)                │
└─────────────────────────────────────────┘
```

### 2. mspan 结构

- 管理固定大小的内存块
- 分配位图跟踪对象状态
- 67种大小类

### 3. 内存分配流程

- 小对象 (<32KB): mcache → mcentral
- 大对象 (>32KB): 直接从 heap

### 4. 内存对齐

- 8字节对齐
- 16字节对齐

## 性能特点

- 分层设计减少锁竞争
- 无锁 mcache 分配
- span 批量管理对象
