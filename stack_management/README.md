# Go 栈管理

## 概述

本项目深入讲解 Go 栈的实现原理，包括连续栈、栈分裂、逃逸分析等核心机制。

## 核心内容

### 1. 连续栈

- 初始大小 2KB，最大 1GB
- 动态扩缩容机制
- 数据整体搬迁

### 2. 栈分裂

- 函数 prologue 检查
- 栈空间不足时扩容
- 2倍扩展策略

### 3. 逃逸分析

- 变量分配位置决策
- 栈 vs 堆分配
- 查看方式：`go build -gcflags=-m`

### 4. 递归优化

- 尾递归
- 迭代改递归
- 栈深度控制

## 栈帧结构

```
┌─────────────────────┐
│   返回地址           │
├─────────────────────┤
│   保存的寄存器       │
├─────────────────────┤
│   局部变量           │
├─────────────────────┤
│   函数参数           │
├─────────────────────┤
│   上一帧 BP         │
└─────────────────────┘
```

## 性能特点

- 无分段栈开销
- 栈空间按需分配
- 逃逸分析优化
